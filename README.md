# Library Demo Application

## How to run the application?

```
git clone https://github.com/adityaburra/library-demo.git
```

```
cd library-demo
```

```
mvn spring-boot:run
```

To run test cases
```
mvn test
```
## Technology stack 
* Spring Boot
* H2 In Memory Database
* Thymeleaf Template Engine
* jQuery
* Maven
